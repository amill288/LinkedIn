Assumptions

Both the standardized residual plot and the qq plot points to residuals being normally distributed without any major outliers. The residuals vs. fitted plot is not a perfectly flat line but there doesn’t appear to be a pattern or funnel effect. The scale location plot also doesn’t have a perfectly straight line, but the data points seem to be equally scattered, which points to homoscedasticity. It doesn’t appear that transforming the data is necessary but because of the small size of the data set, running simulations and removing the doubts of assumption would be beneficial.

Permutation 

set.seed(5555)

perm_run <- replicate(10000, summary(lm(attendance~sample(runs), data=data_runs))$fstatistic[1])

F-stat distribution

Plotting the F stat distribution under the Null hypothesis (no correlation exists), shows us that our observed F-statistic, 5.308, is outside of the permutation distributions 95% CI. This tells us that the permutation distribution agrees that Runs per game and attendance are correlated.

hist(perm_run, freq = FALSE, main = "F value Permutation Distribution", xlab = "F-value", xlim=c(0,8))

abline(v=quantile(perm_run, probs=.95))

abline(v=summary(lm_runs)$fstatistic[1], col="red")

legend(6,.6,  c("95% CI", "Obs. F statistic"), lty=c(1,1), col=c("black", "red") , cex = .8)


Bootstrap 

set.seed(5555)


slope_function_runs <- function(dat, inds) {
  d <- dat[inds,]
  return(with(d,lm(attendance~runs,d)$coefficients[2]))
}
  
boot.data_runs <- boot(data_runs, slope_function_runs, R = 10000)

Bootstrap Slope Estimate

The Bootstrap slope estimate is 8,456.051. This means for every 1 run scored, attendance increases by 8,456.051.
